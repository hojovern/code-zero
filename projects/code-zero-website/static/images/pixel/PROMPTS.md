# Pixel Art Asset Prompts

Use these prompts with an AI image generator (Midjourney, DALL-E, or similar) to create consistent pixel art assets for the code:zero homepage.

## Style Guide
- **Resolution**: 64x64 for icons, 512x256+ for landscapes
- **Colors**: Purple (#4c1d95), Blue (#3730a3), Yellow (#FFD700), Green (#22c55e)
- **Style**: 8-bit/16-bit retro game aesthetic, clean pixel edges, no anti-aliasing

## Generation Tips

1. **For Midjourney**: Add `--v 5 --style raw --no blur, gradient, smooth` to keep pixel edges sharp
2. **For DALL-E**: Specify "no anti-aliasing, sharp pixel edges" in prompt
3. **Post-processing**: May need to resize to exact dimensions and ensure transparency
4. **Consistency**: Generate multiple variations and pick the ones that match the color palette best
