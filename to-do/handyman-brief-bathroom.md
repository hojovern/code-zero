# Contractor Work Order: Master Bathroom Renovation
**Location:** I-Zen Kiara 1 (804 sq ft)
**Scope:** Installation & Finishing Only (Owner provides Toilet, Vanity, Taps, Accessories)

---

## 1. Demolition & Prep
*   [ ] **Remove existing toilet bowl:** careful removal to avoid damaging floor outlet.
*   [ ] **Remove existing vanity cabinet & sink:** dispose of rotting wood material.
*   [ ] **Remove old accessories:** towel racks, toilet paper holders, rusted screws.
*   [ ] **Shower Wall Prep:** Smooth/skim coat the existing cement patch on the shower wall to create a flush surface ready for paint.

## 2. Painting (Epoxy Application)
*   **Material:** Owner will provide/specify Epoxy Tile Paint (White for Walls, Grey for Floor).
*   [ ] **Deep Clean & Sand:** Thoroughly clean tiles to remove soap scum/grease and sand for adhesion.
*   [ ] **Base Coat:** Apply epoxy primer to all wall tiles and floor tiles.
*   [ ] **Finish Coat:** Apply 2 coats of finish epoxy.
    *   *Note:* Ensure shower area is fully cured before water testing.

## 3. Installation (Plumbing & Fixtures)
*   [ ] **Install One-Piece Toilet Bowl:**
    *   Ensure new wax ring/pan collar is properly sealed (no leaks).
    *   Test flush and refill speed.
*   [ ] **Install Wall-Hung Aluminum Vanity:**
    *   Drill and mount securely to wall tiles.
    *   Connect sink waste and trap (ensure no leaks).
    *   Silicon seal the gap between the basin and the wall.
*   [ ] **Install New Taps & Shower:**
    *   Install basin tap (ensure flexible hoses are new/tight).
    *   Install new shower mixer and rail set.
    *   Install bidet spray next to toilet.

## 4. Accessories
*   [ ] **Mount Mirror:** Center above the vanity.
*   [ ] **Install Accessories:** Towel rail, toilet roll holder, robe hooks (locations to be marked by Owner).
*   [ ] **Clean Up:** Remove all debris and old silicon.
