## Morning: Final Preparation

### 1. Final Prep (60 min)

**Technical setup:**
```
You: Help me prepare for the final demo.

Technical checklist:
- [ ] Demo account logged in
- [ ] Browser tabs ready (no embarrassing tabs!)
- [ ] Demo data populated
- [ ] Notifications silenced
- [ ] Browser zoom set (125-150%)
- [ ] Internet connection stable
- [ ] Backup video ready
- [ ] Phone on silent

Demo content:
- [ ] Script reviewed
- [ ] Talking points memorized
- [ ] Timing practiced (5 min max)
- [ ] Transitions smooth
```

**Mental prep:**
- Review your script one more time
- Visualize a successful demo
- Remember: you built something real
- The audience wants you to succeed

### 2. Practice Runs (60 min)

**Practice with your intake:**

Each person does a 5-minute run:
1. Present as if it's real
2. Timer running
3. Others take notes
4. Feedback after each one

**Feedback format:**
- One thing that was great
- One thing to improve
- Any confusing parts

**Refine based on feedback:**
- Cut parts that ran long
- Clarify confusing sections
- Add emphasis where needed

### 3. Dress Rehearsal (60 min)

**Full run-through:**
- Everyone presents in order
- Exact demo format
- Practice transitions between presenters
- Test all tech (screen sharing, etc.)

**Final adjustments:**
- Any last-minute fixes
- Backup plans confirmed
- Order finalized

## Afternoon: Demo Day

### The Format

**5 minutes per person:**
- Strict time limit
- Timer visible
- Moderator will signal 1 minute left
- Cut off at 5 minutes

**Audience:**
- Your intake
- Instructors
- Invited mentors
- Industry guests
- Potential employers/investors

### The 5-Minute Structure

| Section | Time | What to Cover |
|---------|------|---------------|
| Hook | 30 sec | Problem + your solution |
| Journey | 45 sec | What you built over 4 weeks |
| Live Demo | 2.5 min | Show the product working |
| Results | 45 sec | Users, feedback, metrics |
| Close | 30 sec | What's next + thank you |

### Demo Script Template

```
HOOK (30 seconds):
"[Problem statement]

I'm [name], and I built [product name]—[one-line description].

Let me show you how it works."

JOURNEY (45 seconds):
"Four weeks ago, I started with [idea/problem].

Week 1: Built the foundation—AI workspace, landing page, content engine.
Week 2: Added real-time features and automation.
Week 3: Launched to real users.
Week 4: Grew to [X] users.

Here's what it looks like today."

LIVE DEMO (2.5 minutes):
[Share screen, show product]

"This is the landing page. When you sign up..."
[Show signup flow]

"The core feature is [feature]. Watch what happens when I..."
[Demonstrate core feature]

"And here's the AI in action..."
[Show AI feature working]

"Everything updates in real-time..."
[Show real-time if applicable]

RESULTS (45 seconds):
"Since launching:
- [X] users have signed up
- [Y] have used [core feature]
- Average time to value: [Z] seconds

A user told me: '[quote]'

The best part? [Something unexpected you learned]"

CLOSE (30 seconds):
"What's next: [Future plans]

If I could do it again, I'd [one learning].

Thank you. [Product name]—[tagline or one-liner]."
```

### Demo Day Rules

1. **5 minutes maximum** — You will be stopped
2. **Live demo required** — Show the real product
3. **Your URL** — Demo from production, not localhost
4. **Have backup** — Video or screenshots if live fails
5. **Stay calm** — If something breaks, acknowledge and move on
6. **Be proud** — You built this. Own it.

### What Guests Are Looking For

| Looking For | How to Show It |
|-------------|----------------|
| Problem-solving | Clear problem, clear solution |
| Technical ability | Working product, good architecture |
| User focus | Real users, real feedback |
| Initiative | What you did beyond requirements |
| Communication | Clear, confident presentation |
| Potential | What's next, your vision |

### Handling Q&A

**After demos, guests may ask:**
- "What was your biggest challenge?"
- "How would you scale this?"
- "What's your business model?"
- "What would you do differently?"
- "What's next for this product?"

**Tips:**
- Answer honestly
- If you don't know, say so
- Connect back to what you learned
- Show you've thought about the future
