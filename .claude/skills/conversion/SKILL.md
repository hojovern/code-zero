# Conversion Patterns

**Type:** Reference (consult for forms, CTAs, modals, landing pages)

---

## PSYCHOLOGY TRIGGERS

### Urgency
- "Only X spots remaining"
- "Applications close [date]"
- Pulsing dot + live indicator
- Countdown timers (use sparingly)

### Scarcity
- "Limited to 12 per intake"
- "Next intake: [Month]"
- Show spots filled vs available

### Social Proof
- "200+ graduates"
- "94% ship rate"
- Named testimonials with role/company
- Logos of companies (if applicable)

### Risk Reversal
- "Money-back guarantee"
- "No credit card required"
- "Cancel anytime"
- "Free trial / taster session"

### Authority
- "HRDF Claimable" (for Malaysia)
- Certifications, badges
- Media mentions
- Instructor credentials

## CTA PATTERNS

### Button Text (Best → Worst)
1. **Outcome-focused:** "Start Building" ✓
2. **Action + benefit:** "Get Free Session" ✓
3. **Simple action:** "Apply Now" ○
4. **Generic:** "Submit" ✗

### CTA Placement
- Above the fold (visible without scroll)
- After value proposition
- After testimonial/social proof
- Fixed/sticky on mobile for long pages

### CTA Design
- Primary color (green gradient)
- High contrast
- Sufficient padding (min 48px height)
- Arrow icon for forward momentum →

---

## FORM OPTIMIZATION

### Reduce Friction
- Fewer fields = higher completion
- Delay non-essential questions to step 2
- Pre-fill when possible
- Show progress (step 1 of 2)

### Field Priority
1. Email (required - for follow-up)
2. Name (builds relationship)
3. Phone/WhatsApp (optional - faster contact)
4. Everything else (step 2 or post-signup)

### Labels & Placeholders
- Labels above fields (not inside)
- Placeholders show format: "+60 12-345 6789"
- Mark optional fields, not required

### Validation
- Inline validation (as user types)
- Friendly error messages
- Don't clear form on error

## TRUST SIGNALS

### For Forms
```
"No spam. We never share your info."
"Reply within 24 hours"
"[Lock icon] Your data is secure"
```

### For Payments
```
"Money-back guarantee"
"Secure checkout"
"Cancel anytime"
```

### Visual Trust
- Clean, professional design
- Real photos (not stock)
- Named testimonials with faces/initials
- Specific numbers ("94%" not "high")

## MODAL CONVERSION STRUCTURE

```
┌─────────────────────────────────────────┐
│  LEFT: THE SELL          RIGHT: THE ASK │
├─────────────────────────────────────────┤
│                                         │
│  • Headline (outcome)    • Form header  │
│  • Subheadline (how)     • Progress bar │
│  • Value props (3 max)   • Form fields  │
│  • Testimonial           • CTA button   │
│  • Trust badges          • Trust note   │
│                                         │
└─────────────────────────────────────────┘
```

## QUICK WINS

| Issue | Fix |
|-------|-----|
| Low form completion | Remove fields, add progress |
| Low CTA clicks | Change text to outcome, add arrow |
| High bounce | Add social proof above fold |
| Cart abandonment | Add urgency, trust signals |
